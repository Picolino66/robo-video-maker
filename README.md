# Robo-Vídeo-Maker
Projeto open source para que seja feito vídeos automatizados.

# Pré Requisitos
* Node Js
* After Effects
* ImageMagick
* Api: Algorithmia
* Api: Watson
* Google Cloud Plataform
* Api: Custom Search API
* Api: Custom Search Enginer
* Api: YouTube
